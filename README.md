# Yuman — Human Intelligence. AI Amplified.

Consulting website for Yuman, built as a single-page HTML site with bilingual EN/IT support.

## Stack

- Single-file HTML/CSS/JS (no frameworks)
- Google Fonts (Inter)
- Hosted on GitHub Pages

## Local development

Open `index.html` in any browser. No build step required.

## Deploy

This site is deployed automatically via GitHub Pages from the `main` branch.
